# Тестовое задание Лаборатории Касперского

#### Задание
```
Написать скрипт, который получает информацию с сайта threats.kaspersky.com и записывает ее в базу данных. 
Необходимо получить данные об уязвимостях для заданного продукта со страницы https://threats.kaspersky.com/en/product/. 
Скрипт должен быть написан на языке Python, принимать в качестве входного параметра имя продукта (поле Products) и записывать результат в реляционную БД. 
Повторный запуск скрипта не должен дублировать записи в БД. В качестве СУБД использовать sqlite. 
 
Необходимо получить следующие данные: 
Номер статьи (поле Kaspersky Lab ID) 
Заголовок (поле Name) 
Список CVE со ссылками (поле CVE_IDs на странице описания уязвимости, например: https://threats.kaspersky.com/en/vulnerability/KLA11255/)
  
Остальные детали реализации (используемые библиотеки и схема БД) на ваше усмотрение. Просьба не использовать готовые фреймворки по типу django.
```
